#

### Features:
- Live search
- Multiple or sindle search result
- Recent search results
- Viewing movie details
- Fetching loader

### Stack:

- HTML
- CSS + SASS ( SCSS )
- ES6+
- React 
- useReducer + Context for global state
- TMDB API

#

### Install:

```sh
npm install
```
### Run app:

```sh
npm start
```
